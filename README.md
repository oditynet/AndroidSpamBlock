# Описание

Мое первое приложение по блокированию спам звонков. Android умеет показывать во входящих звонках номер телефона и название, если номер в записной книже или была куплена спамерами функция. ПРиложение работает в фоне и входящий звонок сверяет с паттернами,которые вы задали или скачали с обновлением. Минимальный API 33, а это 13 android.
